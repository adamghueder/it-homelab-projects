# Active Directory User Management Lab

## Overview
This lab demonstrates the deployment and management of Active Directory Domain Services in a Windows Server environment. The lab simulates common Help Desk tasks such as user creation, password resets, account unlocks, and group membership management.

## Technologies Used
- Windows Server 2022
- Active Directory Domain Services
- DNS
- Windows Server Manager
- VirtualBox

## Lab Environment
Host Machine:
- Windows 11 Home (23H2)
- Intel i7-1355U
- 32GB RAM

Virtual Machine:
- Windows Server 2022
- 4GB RAM
- 2 CPU cores
- 60GB disk

## Key Tasks Performed
- Installed Active Directory Domain Services
- Promoted server to Domain Controller
- Created Organizational Units (OUs)
- Created users and security groups
- Reset user passwords
- Unlocked locked user accounts
- Managed group memberships
